# 🚀 Preet Patil — AI & No-Code Intern Portfolio

A modern, high-performance **3D developer portfolio website** built with **React**, **Vite**, **Three.js**, **GSAP**, and **WebGL**.

---

## ✨ Highlights

- **3D / WebGL experience** powered by **Three.js**
- Smooth animations with **GSAP**
- Modern **React** codebase
- **AI Chatbot** powered by **Groq API (LLaMA 3.3)**
- Interactive **Chess game** vs AI engine
- Fast, responsive UI (desktop + mobile)
- Deployed on **Vercel**

---

## 🧰 Tech Stack

- **React 18**
- **Three.js / WebGL**
- **GSAP**
- **Groq API (LLaMA 3.3-70b)**
- **Vite**
- **HTML / CSS / JavaScript**

## 🚀 Getting Started

### 1) Clone

```bash
git clone https://github.com/patilpreetviia-netizen/portfolio2.git
cd portfolio2
```

### 2) Install

```bash
npm install
```

### 3) Set up environment variables

Create a `.env` file in the root directory:

```bash
GROQ_API_KEY=your_groq_api_key_here
```

Get your free API key at: https://console.groq.com

### 4) Run locally

```bash
npm run dev
```

### 5) Build

```bash
npm run build
```
